# Datos del mapa de evolución

- `americas.geojson`: selección de países de Natural Earth, escala 1:50m. Natural Earth declara estos datos de dominio público y permite reutilización comercial.
- `mexico-states.geojson` y `chihuahua.geojson`: geometrías simplificadas de geoBoundaries `MEX-ADM1-31927357`, cuya fuente indicada es INEGI y licencia CC BY 3.0 IGO.
- `us-states.geojson`: geometrías simplificadas de geoBoundaries `USA-ADM1-66186276`, cuya fuente indicada es United States Census Bureau y es de dominio público. Sólo incluye EE. UU. continental.

Ambos archivos se sirven localmente. El crédito se muestra dentro del mapa.

Las coordenadas de los puntos de presencia son las aprobadas para Ecosat.
